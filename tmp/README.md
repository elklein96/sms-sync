# sms-sync

## What is this?

This is a web interface for sending SMS messages through an Android phone.

## Building the app

1. Clone this repository.

	```
	$ git clone https://github.com/elklein96/sms-sync
	```
2. Install some dependencies.
	
	```
	$ npm install && bower install
	```

3. Run the app.

	```
	$ grunt
	```
