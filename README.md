# chat-app

## What is this?

This is a simple chat application that employs web sockets. 

## Building the app

1. Clone this repository.

	```
	$ git clone https://github.com/elklein96/chat-app
	```
2. Install some dependencies.
	
	```
	$ npm install && bower install
	```

3. Run the app.

	```
	$ grunt
	```
